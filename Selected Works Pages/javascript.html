Obsticles To Feeding Pacman

A game created for my Global Food Systems class in Spring of 2024. This is a game inspired by pacman created to teach student about the obsticles to feeding 9.5 billion people. It taught me concepts of human-computer interaction, Iterface design, collisions/triggers, and sprites.

Murder Mystery

In collaboration with Alyssa Mazzone and Danika Perez:

In this murder mystery game, the user must play several minigames in order to collect hints to find the murderer. This game uses concepts of On-screen Dynamics, Random Numbers, Text, and particle systems.

For on-screen dynamics, the door and object respond to each other. If the user clicks one door, the others close. If the user chooses the correct door, the continue button appears. In the end, the user must decide on the murderer. This can trigger a winner or loser screen and corresponding sound effects.

Random numbers are also utilized. Every time the user plays the game, the murderer is chosen at random. In the first minigame, the weapons are moving in a random direction. In the second minigame, the weapon is placed under a randomly chosen door.

Text is also used across the game. On the first screen, we have included Text that flashes, changing in real time. In the first minigame, the text color changes as a clue to who the murderer is.

Lastly, we use a particle system in the first minigame. The weapons have a velocity and position. If they collide with each other or the walls, they bounce off of each other.
